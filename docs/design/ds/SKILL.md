If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Key rules: two palettes, one system — indigo app / slate+violet+amber web; amber never enters the app. Suggestions are chips phrased as questions that never act alone. Numbers always tabular. Sentence case, bilingual ES/EN, honest copy ("Measured, not promised"). Motion is only breathe (3.6s) and pulse (1s), behind prefers-reduced-motion.
